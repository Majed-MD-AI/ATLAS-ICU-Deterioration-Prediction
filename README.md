# ATLAS - ICU Deterioration Prediction

A simple machine learning project aimed at predicting early signs of patient deterioration in the ICU using basic clinical features.

---

## 🧠 Project Overview

This project explores how machine learning can be applied to detect early warning signs of patient deterioration in critical care settings.

Using a small dataset with basic vital signs such as:
- Age
- Heart Rate
- Blood Pressure

Two models were implemented:
- Logistic Regression
- Random Forest

The goal is not just prediction, but understanding how simple models can contribute to clinical decision support systems.

---

## 🏥 Clinical Perspective

In real-world ICU environments, early detection of deterioration can significantly impact patient outcomes.

This project reflects a simplified version of a much larger vision:
Building intelligent systems that assist clinicians in making faster, safer, and more informed decisions.

---

## ⚙️ Implementation

The workflow includes:
- Data preparation using pandas
- Splitting data into training and testing sets
- Training models using scikit-learn
- Making predictions on new patient data
- Evaluating performance using accuracy

---

## 📊 Results

- Logistic Regression accuracy: **1.0**
- Random Forest accuracy: **0.0**

**Note:** Due to the very small dataset, results may not generalize well.  
This project is intended as a conceptual demonstration rather than a production-ready model.
